# Hardware-interface libraries — installed in the container / on the Pi only.
# Kept OUT of requirements.txt so `just test` and laptop dev stay clean (Blinka's
# `import board` expects a real Pi; the app imports these lazily in factory.py).
#
# Digital sensors (I²C + 1-Wire) — what the POC actually deploys:
adafruit-blinka                 # board/busio for the I²C bus
adafruit-circuitpython-hdc302x  # HDC3022 temperature + humidity
adafruit-circuitpython-bh1750   # BH1750 ambient light
w1thermsensor                   # DS18B20 soil-temperature probes (1-Wire)
lgpio                           # REQUIRED by Blinka's board module on the Pi 5 (bcm2712),
                                # even for pure I²C. No Debian pkg / wheel — the lg C lib
                                # is built from source in the Containerfile.
#
# Deferred until the resistive soil sensor / pump actuation phase:
